
👩‍🏫 검색
1. 게시글 목록 페이지 - 검색창 UI 

2. 검색을 하기 위한 쿼리 수정
    ``` 
        SELECT *
        FROM board
        WHERE title LIKE '%keyword%'
    ```
3. 검색 옵션
    - 전체      : 0
    - 제목      : 1
    - 내용      : 2
    - 제목+내용 : 3
    - 작성자    : 4
